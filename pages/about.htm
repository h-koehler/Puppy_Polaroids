Puppy Polaroids is a mock website created using HTML, CSS and JS. It features an interactive cover page with hoverable and clickable photos. On hover, the photos will flip over to show a high-level information card regarding the dog breed. On click, a modal box will open, displaying a more in-depth description of the breed. Both the cover page and modal boxes have a responsive layout that adjusts the navigation bar and layout according to the screen size.
